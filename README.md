# Jogo do Número Secreto

# Adivinhe o número secreto gerado pelo sistema
-------------------------------------------------------------------------------------------

# Objetivo:
Aplicar conceitos de JavaScript e lógica de programação, como:

* Variáveis
* Funções
* Condicionais

# Tecnologias utilizadas:

* HTML5
* CSS3
* JavaScript
* Git e GitHub

# Lógica do projeto:

* O sistema gera um número secreto aleatório
* O usuário tenta adivinhar o número através de um input
* A cada tentativa, o sistema informa se o número é maior ou menor
* O jogo continua até o usuário acertar
* Ao acertar, é exibida uma mensagem de sucesso
* Automaticamente a contagem é reiniciada ao clicar em "novo jogo".

# Funcionalidades:

* [x] Geração de número aleatório
* [x] Validação de tentativas
* [x] Feedback ao usuário (maior ou menor)
* [x] Contagem de tentativas
* [x] Reinício do jogo

# Exemplo de saída:

O número secreto é maior
O número secreto é menor
Parabéns! Você acertou!
